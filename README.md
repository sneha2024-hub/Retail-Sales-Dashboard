# 📊 Retail Sales Dashboard (Power BI)

## 🚀 Project Overview
This project showcases an interactive Retail Sales Dashboard built using Power BI.  
The goal was to analyze sales performance and provide insights across products, regions, and time.

---

## 📌 Key Features
- 📈 KPI Cards: Total Sales, Quantity, Profit, Profit Margin  
- 📊 Trend Analysis with monthly performance  
- 🛍️ Top 10 Products by Sales  
- 🌍 Sales distribution by Region and State  
- 🏬 Sales breakdown by Store Type  
- 🎛️ Interactive Year slicer for dynamic filtering  

---

## 🔍 Key Insight
Sales peaked during mid-year, with specific product categories contributing significantly to revenue growth.

---

## 🛠️ Tools & Technologies

- Data Modeling
